- 모바일 고정 폭 UI
- PWA manifest / 서비스 워커 / 앱 아이콘
- 세이브 버전과 0.1 데이터 마이그레이션
- 스테이지/지역 해금 진행
- 최초 클리어 보상과 반복 보상 분리
- 속성 상성, 치명타, 전투 속도, 전투 로그
- 일일 미션과 가챠 천장
- 보스 스테이지와 보스 전용 문구/보상
- 지역/스테이지 스토리 텍스트
- Lv.(n) / 구매 / 일반 / 이벤트 / 특별 / DEVELOPER BADGE 업적 구조
- 인벤토리 필터와 HOME 진행 가이드
- EX 컷인, 등급별 가챠 연출, 최소 효과음
- 파티 편성, 패시브, 조합 보너스
- 화상/감전/감속 상태 이상과 코어 버스트
- 주간 미션과 보스 반복 파밍 재료
- 전체 속성표와 속성 형판 합성/개봉
- ExT / Mind / Deep Mind 성장 기반
- 재화, 광물, 보석, 아이템 도감
- 기본 추첨 확률표와 확장 캐릭터 풀
- 피온스 / 트로맨션 / 오로시스 지역
- 10개 스테이지
- 자동 평타 + 수동 EX 전투
- 쿼츠 / 레드블링 / 블루블링 / 형판 조각 등 재화 도감
- 캐릭터 4명과 강화
- 추첨 1종
- 인벤토리와 localStorage 저장

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

## GitHub Pages 배포

1. 이 폴더를 GitHub 저장소에 올립니다.
2. 저장소의 Settings > Pages로 이동합니다.
3. Source를 Deploy from a branch로 설정합니다.
4. Branch는 main, 폴더는 root를 선택합니다.
5. 배포 URL에 접속한 뒤 모바일 브라우저에서 홈 화면에 추가합니다.

서비스 워커는 HTTPS 또는 localhost에서만 동작합니다. GitHub Pages 배포 후에는 PWA 캐시와 홈 화면 추가가 정상 적용됩니다.
